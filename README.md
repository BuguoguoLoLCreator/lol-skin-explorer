# [布锅锅联盟宇宙丨原画](https://splash.buguoguo.cn)

你是否曾想查看某位英雄的皮肤原画，却又不想特意打开游戏客户端？又或者只是想轻松地浏览某个特定的皮肤宇宙？现在，这一切都变得轻而易举！

## 功能亮点

* 与 PBE 测试服同步更新！（抢先查看尚未上线的皮肤。）
* 皮肤按英雄、系列和宇宙分类整理。
* 可浏览旧版皮肤原画（最早可追溯到 7.1 版本）。
* 支持添加至 Safari（iOS）或 Chrome（Android），体验更沉浸。
* 支持键盘快捷键与手势操作，体验更流畅。
* 页面加载速度极快。
